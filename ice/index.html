This page collects all of my ICE (In‑Class Exercise) challenges for WDD130. Each section describes what I worked on and what I learned.

ICE Assignments

  • ICE 01 – Basic HTML Structure

    In this challenge I used my knowledge of CSS and using it to how to move and position elements on a page. I learned how to use the CSS properties of top, bottom, left, right, and z-index to position the elements on the page.

    View ICE 01

  • ICE 02 – CSS Styling

    I learned how to connect a CSS file and style text, colors, and simple layouts. The webpage allows you to view ideas for Father's day gifts and meals. I used a combination of HTML and CSS to create a visually appealing layout with images and text.

    View ICE 02

  • ICE 03 – Images and Layout

    In this challange I created a webpage that demonstrates basic HTML and CSS structures. Created a card that displays a image of the Christus with the first presidency and twelve apostles centered, each with a boarder.

    View ICE 03

  • ICE 04 – Forms and Input

    In this challenge I created a webpage that demonstrates basic HTML and CSS structures. Created a form that allows users to input their name, email, and allow them to subscribe to a newsletter. The form includes validation to ensure that the user has entered all required fields.

    View ICE 04

  • ICE 05 – CSS Grid

    In this challenge I created a webpage that demonstrates basic HTML and CSS structures. Created a webpage that uses CSS grid to create the flags of different countries. The flags are created using a combination of HTML and CSS, and are displayed in a grid layout.
